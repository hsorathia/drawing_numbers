Number Recognition

Neural network application to identify user drawn numbers

About

Users will draw and save a number onto our drawing canvas.


Once saved, the image will be sent to the backend for number prediction.


The image will be converted into a 28x28px image and converted into readable data.


Data will be fed into our neural network and output the prediction onto the frontend.

Neural Network

Our neural network is constructed from scratch using NumPy


Along with the MNIST dataset to train our network.


1 Input Layer
2 Hidden Layers
1 Output Layer


Feedforward function for dataset
Split training data into batches
Backwards propagation to adjust values
